# Applications Guide

This subarea stores process documents for external applications, including preparation notes, answer drafts, and submission checklists.

## What Belongs Here

- application prep outlines and requirement notes
- answer drafts and revision notes
- submission checklists and timeline planning

## Language Rule

- All tracked documents in this folder must be written in English.
- Keep off-repository brainstorming outside tracked files unless and until it has been normalized into English.

## Boundary Rule

- Keep organization-specific prep materials here when they are process artifacts.
- If a document becomes a reusable analytical asset, move or duplicate the stable version into `research/core/` or the relevant extension track.
- This folder is not an approved bilingual legacy reference cluster.
