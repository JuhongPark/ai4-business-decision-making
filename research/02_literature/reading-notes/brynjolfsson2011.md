# Reading Note

source_id: brynjolfsson2011
title: Strength in Numbers: How Does Data-Driven Decisionmaking Affect Firm Performance?
author: Erik Brynjolfsson, Lorin M. Hitt, Heekyung H. Kim
year: 2011
type: conference paper
link: https://aisel.aisnet.org/icis2011/proceedings/economicvalueIS/13/

## Summary

This study examines whether firms that emphasize data-driven decision-making perform better than comparable firms. It finds a positive association between data-driven practices and firm performance measures.

## Key Arguments

- Data-driven decision-making is linked to stronger productivity and output.
- Analytical capability creates value when embedded in organizational practice.
- Better outcomes depend on how firms use data, not simply on having more data.

## Evidence

Using survey data from 179 large publicly traded firms, the authors report that firms using data-driven decision-making show output and productivity levels that exceed expectations based on other investments and IT use.

## Relevance to Research Question

This source helps connect analytical decision support to organizational performance. It supports the decision to study firm-level outcomes rather than only technical accuracy.

## Useful Quotes

- Data-driven decision-making should be treated as a managerial practice with measurable performance implications.

## My Notes

This paper is useful as a bridge between analytics capability and performance effects. It also supports the argument that AI value depends on organizational use conditions.
