# Expansion Plan Review

reviewer: Priya Raman
target: research/08_expansion/phase-08-expansion-plan.md
overall_assessment: promising_but_should_bias_toward_high-stakes_rigor

## Strengths

- The plan correctly recognizes that evidence quality is now the bottleneck.
- The proposed evidence hierarchy is necessary and overdue.
- The emphasis on governance as a condition of performance remains strong.

## Concerns

### Concern 1: High-stakes domains deserve priority over broad balance

If the study is expanded, finance and HR should receive disproportionate attention because these domains create the strongest need for defensible governance claims.

### Concern 2: Investigative reporting should be clearly tiered below regulator and primary sources

The plan mentions this but should institutionalize it. The next version should define evidence tiers and rules for when lower-tier sources are acceptable.

### Concern 3: The framework should express default restriction logic

The next-stage framework should not only say when AI fits. It should also state default restrictions for sensitive domains unless specific controls are demonstrated.

## Recommendations

1. Prioritize finance and HR as test domains for the next-stage revision
2. Create formal evidence tiers
3. Add explicit restriction rules for high-impact decisions
