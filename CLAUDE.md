# Project Rules

## Commit Authorship
- All commits MUST be authored solely by **Juhong Park <juhongp@mit.edu>**
- Do NOT include `Co-Authored-By` trailers in commit messages
- Do NOT add any other author or co-author information to commits

## Language Rules
- Write all commit messages in English.
- Write all document body content in English.
- Do not include Korean text in tracked repository content unless the user explicitly requests it for a specific file.
- If Korean text is committed by mistake, rewrite the affected content and, if necessary, rewrite the relevant git history so the committed version is English.
